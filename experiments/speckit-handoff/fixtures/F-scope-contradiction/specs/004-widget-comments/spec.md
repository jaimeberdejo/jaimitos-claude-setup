# Feature Specification: Widget Comments

## Requirements

### Functional Requirements

- **FR-001**: System MUST let a visitor post a comment on a widget detail page.
- **FR-002**: System MUST display comments newest-first under the widget.
- **FR-003**: System MUST let a visitor share a widget to their followers.

### Success Criteria / Measurable Outcomes

- **SC-001**: A posted comment appears on the page within 1 second.
- **SC-002**: A widget page with 500 comments renders in under 800 ms.
